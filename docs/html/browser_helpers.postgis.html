browser_helpers.postgis module¶

Summary of postgis.py¶

Use the database connections defined in postGIS_tools.get_postGIS_config() to add DB connections under PostGIS within the QGIS Browser.

browser_helpers.postgis.add_postgis_connections(dbs_to_ignore: list = ['defaultdb', '_dodb'], remove_existing: bool = True) → None[source]¶

Add a database connection for all DBs that exist on the clusters defined in the user’s config.txt

For more details, see postGIS_tools.get_postGIS_config()

Parameters

  • dbs_to_ignore – list of databases to skip creating entries for

  • remove_existing – if True, start by removing the user’s DB list

browser_helpers.postgis.remove_postgis_connections() → None[source]¶

Remove all entries under PostGIS in the Browser
